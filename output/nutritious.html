nutritious

Category: ラテン語系

いい単語を選びましたね！nutritious は、健康・食事・生活習慣に関する文脈で非常によく使われる形容詞です。語源もラテン語由来で、似た語との違いもおさえておくと語感の鋭さがアップします。

🔤 単語：nutritious（形容詞）

1. 🧠 意味

意味 用法・例文
栄養のある、栄養価の高い This soup is very nutritious.（このスープはとても栄養がある）
  • ✅ 「体に良い」「成長や健康に役立つ栄養を含む」という前向きな意味で使われる

2. 🧬 語源・由来

  • ラテン語：nutrire（養う、育てる）

    • → nutritio（栄養）＋ -ous（形容詞を作る接尾辞）

🔑 「育てる（nurture）」とも同じ語源！
→ 体を“育てる”食べ物 → 栄養がある！

3. ✏️ 用法と例文

✅ 健康的な食事の文脈で

  • Children need nutritious meals to grow strong.

  • Fruits and vegetables are nutritious and low in calories.

✅ 肯定的な評価として

  • A nutritious breakfast helps you start the day right.

4. 🧠 覚え方のコツ

  • 「ニュートリション＋ous（～な）＝栄養のある」

  • 「ニュートリ系＝健康そうなやつ」と覚えると感覚的にも定着！

5. 🔁 同義語・類語比較

単語 意味 フォーマル度 ニュアンス・使い方
nutritious 栄養のある ★★★★☆ 健康・科学的視点、ポジティブな評価
healthy 健康に良い ★★☆☆☆ より広く、生活習慣・運動にも使える
wholesome 健康的・健全 ★★★☆☆ 栄養＋心の健やかさ、古風・温かい
nourishing 栄養を与える ★★★☆☆ 心身にしみこむような栄養感、文学的
nutritional 栄養に関する（形容詞） ★★★★★ 科学・学術・ラベル向け

📌 nutritious は「実際に栄養価が高いこと」
📌 nutritional は「栄養に関する（形容詞化）」→ nutritional value など

6. 🌍 ラテン語 or ゲルマン語？

  • ✅ ラテン語由来（nutrire） → 科学・教育・健康関連において、フォーマルかつ洗練された響きがある

7. 🌱 同じ語根の単語：nutr-/nour-（育てる・栄養）

単語 品詞 意味 備考
nutrition 名詞 栄養、栄養学 最も基本的な語
nutritional 形容詞 栄養に関する フォーマル・学術的
nourish 動詞 栄養を与える、育む 文学的にも使われる
nurture 動詞・名詞 育てる、育成 教育・子育て文脈でも頻出

8. 🧾 似た綴りの単語との違い（派生語でないもの）

単語 意味 注意点
nutritious 栄養がある ポジティブな栄養価表現
ambitious 野心的な 似た「-tious」語尾だが全く意味は異なる
suspicious 疑わしい -cious/-tious 系の見た目に騙されないこと！

9. 🔍 フォーマル度マップ

Healthy Wholesome Nutritious Nutritional | | | | casual ----------------------------------> formal ↑日常 ↑温かい ↑健康食品 ↑学術的

✅ まとめ

  • nutritious = 「栄養価が高くて、体に良い」

  • フォーマルだが日常会話・広告・教育現場でも広く使われる

  • 同根語（nutrition, nourish, nurture）と一緒に覚えると語彙が広がる

  • healthy や wholesome とのニュアンスの違いを意識しよう！
