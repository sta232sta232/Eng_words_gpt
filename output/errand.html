errand

Category: ゲルマン語系, ラテン語系

いいですね！日常でも物語の中でもよく出てくる単語、errand。短いのに、古英語・ゲルマン語にルーツを持つ、深みのある言葉です。使い方・同義語との違い・語源・覚え方まで、がっつり解説します！

🔤 単語：errand（名詞）

1. 🧠 意味

意味 用法・例文
使い・用事・おつかい I have to run an errand for my mom.（母の用事でおつかいに行かないと）
使命・任務（古風・文語的） He was sent on a secret errand.（彼は秘密の任務に送られた）
  • ✅ 通常、短時間で終わる用事（買い物、配達、メッセージ伝達など）を指します。

2. 🧬 語源・由来

  • 古英語：ǣrende（メッセージ、使命）

  • ゲルマン語系：「ある目的のための行動」

🔑 元々は「用事を託されること」「使命を帯びた行動」という意味。

3. ✏️ 用法と例文

✅ 日常的な「用事・おつかい」

  • I need to run a few errands this afternoon.

  • Can you do me a quick errand?

✅ メッセージ・使者的な（やや古風）

  • He was on an errand of mercy.（慈善の使命を帯びていた）

  • A knight was sent on an errand by the king.

4. 🧠 覚え方のコツ

  • 「用事エラーんど（errand）」→ 誰かに頼まれてやること＝用事

  • 「えらい任務 → errand」：昔は「使命感」強め！

5. 🔁 同義語・類語比較

単語 意味 フォーマル度 ニュアンス・使い方
errand おつかい・使命 ★★☆☆☆ 短時間の使い走り、やや古風でもOK
task 任務、作業 ★★★☆☆ より広い意味（仕事・課題）
chore 雑用、日常の面倒事 ★☆☆☆☆ 家事・いやいやすること
mission 任務、目的 ★★★★☆ 大きめ・正式・目的が明確
assignment 割り当てられた任務 ★★★★☆ ビジネス・学業でよく使う
delivery 配達 ★★☆☆☆ errand の一部目的になることも

🎯「用事の小ささ・日常性」がポイント
→ それが errand のコアイメージ！

6. 🌍 ラテン系 or ゲルマン系？

  • ✅ ゲルマン語系（古英語）
    → 日常的、カジュアル、あたたかみのある響き
    → 「任務」という意味を持つのに、堅苦しくない！

7. 🌱 同じ語根の語（err-）との違い

単語 意味 関連性
errand 使い、使命 直接語源では「古英語由来」
error / err 間違い、過つ ラテン語 errare（さまよう）由来で、語源が違う

🛑 注意：**error（エラー）**とスペルが似てるけど、全くの別語源！

8. ✍️ フォーマル度マップ（視覚化）

Flyer Chore Errand Task Assignment Mission | | | | | | informal ---------------------------→ formal ↑（日常で頻出）

✅ まとめ

  • errand は「短い用事、使命」→ 誰かのためにちょっとしたことをやる

  • 古英語・ゲルマン語由来 → 温かみのある、カジュアルだけど深みもある言葉

  • task / mission / chore / assignment との違いを意識！

  • 「エラーんど」ではなく、「頼まれ事（使命感）」と覚えるのがコツ！
